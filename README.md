# Mars rover

Implementation of the (Google Mars Rover challenge)[https://code.google.com/archive/p/marsrovertechchallenge/] in go. 

## Requirements

- Docker

## Run

```bash
./go.sh run main.go
./go.sh run main.go input/challenge.txt
```
